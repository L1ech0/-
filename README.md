# Jystreporst
一个适用于mac简单的渗透报告生成器<br>
偷懒神器<br>
适用于等保测评渗透测试<br>
使用方法<br>
先安装所需要的库<br>
pip install -r requirements.txt<br>
输入命令打开gui<br>
python STreport.py<br>
